# 🌍 TravelXplore

TravelXplore is a modern and responsive travel landing page designed to inspire users to explore new destinations. The project features a clean interface, attractive design, and an easy-to-use navigation experience.

## ✨ Features

- 🏠 Responsive Navigation Bar
- 🌄 Beautiful Hero Section
- 🖼️ Travel Gallery
- 🧳 Services Section
- 📞 Contact Section
- 💻 Clean & Responsive Design
- ⚡ Fast and Lightweight

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
TravelXplore/
│── index.html
│── landing_page.css
│── README.md
```

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/TravelXplore.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.
